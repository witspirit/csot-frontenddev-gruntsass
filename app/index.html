Workshop tasks:

Grunt

  • Open “workshops/2. Grunt and Sass” as a project in your WebStorm.
  • Create a package.json file
  • Create a bower.json file
  • Import the sass bootstrap file in your “main.scss” file:
    • path: “bower_components/bootstrap-sass-official/assets/stylesheets/_bootstrap.scss”
  • Download the following grunt tasks and save the dependencies to package.json:
    • clean, copy, sass compiler, uglify, cssmin, watch
  • Add these tasks to your Gruntfile:
    • Clean your “dist” folder
    • compile your Sass file to CSS
    • minify JavaScript & CSS files and put them in your ‘dist/scripts’ and ‘dist/styles’ folder.
    • copy your “index.html” page and “bower_components” folder into your “dist” folder
    • JSHint your JavaScript file
    • watch your index.html page, javascript files, scss files (compile to css when changed)
    • use grunt watch live reloading when changing index.html, js, scss files
      • more info: https://github.com/gruntjs/grunt-contrib-watch#live-reloading
    When you execute the “grunt” command, the index page from the “dist” folder should be styled with bootstrap.
  • Test if your website refreshes when you change your index.html, js or scss file.

Sass

--> Use nesting to have some structure in your file

  • Create a variables.scss file in your styles folder
  • Add 3 color variables and 1 font variable to this file:
    • primary-color: #CC8700
    • secondary-color: #997A3D
    • tertiary-color: #FF6700
  • header-font: Myriad Pro
  • Import this variables.scss in your main.scss
  • Change the blue color of all links (<a>) in your header to your primary-color.
  • Change the font in your “jumbotron” to your header-font and align this text horizontally in the center.
  • Give the “jumbotron” a margin-bottom of 5px.
  • Give the active menu item (in this case home) a secondary-color.
  • Create file mixins.scss in your styles folder and import it in main.scc
  • Create a mixin in this file for a crossbrowser transition with the transition argument as parameters.
    • Use this mixin to change the text color to your tertiary-color in your jumbotron when your hover over the jumbotron.
  • Create a new div with class “math” between the “jumbotron” div and the “row tasks” div with the following properties:
    • Height: 10 times smaller then the navbar ($navbar-height variable). Use a mathematical function to calculate this.
    • background-color: tertiary-color
